상속 - 엔진에 있는 특정 오브젝트 뿐만 아니라 엔진에 필요한 모든 기능을 사용할 때 끌어와서 사용

컴포넌트 - 주로 오브젝트를 만들 때 사용

빈 오브젝트에 다양한 컴포넌트를 조립 조합해서 새로운 기능을 추가해서 오브젝트를 부여하는 것

기획자는 원하는 기능을 제공하는 컴포넌트를 찾아낼 수 있어야 함

게임 엔진에서 제공하는 컴포넌트들이 무엇을 제공하는지 찾아내는 것 중요

클래스를 어딘가에서 상속을 받아 만드는 것 가능 
	ㄴ그렇게 만들어진 클래스 및 컴포넌트들을 조합을 해서 오브젝트 생성 가능 

굉장히 독립되어 있는 하나하나의 컴포넌트, 클래스들을 만드는 것이 중요

상속과 컴포넌트 패턴 둘다 사용


그 컴포넌트가 어딜 참조하고있는지 잘 체크가 되지 않으면 해당 컴포넌트는 작동x

컴포넌트 하나가 에러가나는 것이지, 오브젝트 전체가 에러가 나지 않음 - 해당 컴포넌트만 수정해주면됨
